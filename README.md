## Sobre o projeto

Este e um clone do classico Agar.io feito em um unico arquivo HTML com JavaScript puro e Canvas API — sem frameworks, sem dependencias externas. O jogo foi criado ao vivo no canal usando o **Claude Code** como assistente de programacao.

---

## Como jogar

| Controle | Acao |
|---|---|
| Mouse | Mover a celula |
| `Space` | Dividir a celula |
| `W` | Ejetar massa |

**Objetivo:** Coma bolinhas e celulas menores para crescer. Fuja de celulas maiores. Domine o placar!

---

## Elementos do jogo

- **Bolinhas coloridas** — comida espalhada pelo mapa, colete para crescer
- **Virus verde** (espinho) — divide celulas grandes que passarem por cima; celulas pequenas ficam protegidas dentro dele
- **Spike vermelho** — gera comida ao redor periodicamente
- **Bots com IA** — 100 bots com comportamento de fuga, caca e busca por comida
- **Minimapa** — visao geral do mapa no canto inferior esquerdo
- **Placar** — top 10 do servidor em tempo real
- **Recorde pessoal** — salvo localmente no navegador
